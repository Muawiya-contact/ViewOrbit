# ViewOrbit Frontend

Developer instructions for running the Next.js frontend locally.

## Prerequisites
- Node.js 18+
- npm 9+

## Setup
```bash
npm install
```

## Local Environment
Create or verify:
```
.env.local
NEXT_PUBLIC_LOCAL_MODE=true
```

## Run Locally
```bash
npm run dev
```

Open http://localhost:3000

## Build
```bash
npm run build
npm run start
```

## Local Mode Seed
- Admin account: admin@vieworbit.local / admin123
- Demo tasks are auto-seeded for lifecycle testing.
